# Football-project
Football League Manager
Описание на проекта

Проектът представлява информационна система за управление на футболна лига.
Системата позволява съхраняване и управление на информация за футболни клубове, играчи и мачове между тях.
Целта на проекта е да се създаде структурирана база данни, чрез която да може да се добавя, редактира, изтрива и преглежда информация за футболните отбори и техните резултати.
Проектът решава проблема с хаотичното съхранение на данни, като организира информацията в релационна база данни.

Основни функции на системата

Добавяне на нов клуб
Редактиране на клуб
Изтриване на клуб
Добавяне на играч към клуб
Добавяне на мач
Преглед на списък с всички клубове
Справка за изиграни мачове
